reviews should be going to reviews.co.uk



4 star and 5 star should be using the reviews.co.uk's "dynamic link", click to see example:


http://www.reviews.co.uk/store/landing_new_review?store=cornerstone&user=Di%20Su&order_id=251626&email=di%40cornerstone.co.uk&products=SILD8X;DEO&type=combined&rating=5


1. store = leave this as cornerstone

2. user = full name of user

3. order_id = generate a random number

4. email = email address of user

5. products = we assign a SKU in reviews.co.uk to every product, see product key

6. type = leave this as combined

7. rating = number to represent stars out of 5. so we will have "4" for the 4th star, and "5" for the 5th star
